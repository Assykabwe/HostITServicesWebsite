HOST IT SERVICES WEBSITE
Project Overview

This is a custom-built e-commerce web application designed for selling web hosting services, domains,
and other digital services. The system allows users to browse services, register or transfer domains,
add items to a cart, and proceed to checkout. The platform supports both guest users and registered
users, with full cart and wishlist management.
Key features include:
- Dynamic service and domain selection
- Session-based cart management for guests
- Persistent cart for logged-in users
- Quantity and option (size/color/domain) selection
- Wishlist functionality
- Review & Checkout section with dynamic total calculation
- Responsive design for desktop, tablet, and mobile
Features
User Features
- Browse available services and products.
- Add services and domains to cart.
- Adjust quantity of services in cart.
- Remove items from cart.
- Review & checkout section displaying product details and total.
- Register new domains or transfer existing ones.
- Persist wishlist across sessions for logged-in users.
- Smooth navigation across sections: Web Hosting, Domain Registration, Checkout, etc.
- Responsive layout compatible with mobile and desktop.
Admin Features
- Add, update, and delete services/products.
- Manage Response Ticket System.
- Manage categories including combined categories
Technologies Used
- Frontend: HTML5, CSS3, JavaScript (Vanilla JS)
- Backend: PHP
- Database: MySQL
- Libraries: SweetAlert for alerts
